# Git Bash

## Zijn er changes?

```console
git status
```

### Changes pushen

Om code te pushen moet je minstens volgende stappen volgen:

1. Changes stagen
2. Commit maken
3. Commit pushen

#### Changes stagen

```bash
git add . ( '.' voor hele directory)
```

#### Commit maken

```bash
git commit -m "mijn beschrijving"
```

#### Commit pushen

```bash
git push
```
