# Cheatsheet: Vagrant

| Task                  | Command             |
| :-------------------- | :------------------ |
| Initialize repository | `vagrant init`      |
| Current vm status     | `vagrant status`    |
| Start vm              | `vagrant up`        |
| Stop vm               | `vagrant halt`      |
| Delete vm             | `vagrant destroy`   |
| provision vm          | `vagrant provision` |
